# search-UI

A search component for countries and capital cities.

## This projects uses the following
- webpack
- React
- SCSS
- Babel
- Axios

## Getting started

Install:

```bash
$ cd setup
$ npm install
```

Dev server:

```bash
$ cd setup
$ npm run build:devshow
```

Dev build:

```bash
$ cd setup
$ npm run build:dev
```
Production build:

```bash
$ cd setup
$ npm run build:dist
